## Customer Funnel & Conversion Dashboard

A marketing analytics dashboard built with Python and Streamlit.

### What it shows
- Conversion funnel from visitors to purchases
- Weekly trend by traffic source
- Conversion rate comparison across Organic, Paid Ads, Social

### Tools used
Python, Pandas, Plotly, Streamlit

### How to run
pip install pandas plotly streamlit
streamlit run dashboard.py
